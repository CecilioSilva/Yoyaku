# Yoyaku

Yoyaku is an app to sort and make your preorders more clear.

![Overview](https://i.imgur.com/Zk7urrv.jpg)

    
## Features

- Easy to use
- Fast
- Ofline mode
- Backups
- Notifications

#### Requirements
 - Android 11 and up
